Java IDEA
  1. 删除项目的 .idea 目录
  2. 检查 Maven 的项目结构在 IDEA 中是否标记正常
    • src/main/java 是否标记为 源代码根目录
    • src/test/java 是否标记为 测试源代码根目录
    • src/main/resources 是否标记为 资源根目录
    • src/test/resources 是否标记为 测试资源根目录
  3. 检查 JDK 设置是否一致
