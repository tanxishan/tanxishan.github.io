Node.js

Windows 使用 scoop

  1. 卸载所有Node.js安装,并删除路径C:\Program Files (x86)\nodejs和C:\Program Files\nodejs

  2. 下载nvm-noinstall

  3. 解压到希望安装的目录D:\ProgramFiles\nvm-noinstall

  4. 设置环境变量NVM_HOME为D:\ProgramFiles\nvm-noinstall，NVM_SYMLINK为D:\ProgramFiles\nvm-noinstall\Node.js(Node.js应该是不存在的目录)

  5. 将环境变量%NVM_HOME%和%NVM_SYMLINK%加到Path中

  6. 添加配置文件D:\ProgramFiles\nvm-noinstall\settings.txt

  7. nvm基本使用

  8. 切换到具体Node.js版本后
